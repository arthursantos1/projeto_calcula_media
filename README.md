# projeto_calcula_media

<p>Projeto realizado durante o curso da EBAC front-end do modulo 8</p>
<p>Foi utilizado no projeto HTML, CSS e JavaScript</p>
<p>Calculadora de média de notas com interface intuitiva. Permite:</p>
<ul>
  <li>Inserir notas</li>
  <li>Visualizar o nome da atividade e a nota</li>
  <li>Calcular a média</li>
  <li>Resultado da média e também se foi aprovado ou não</li>
</ul>
